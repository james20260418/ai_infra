# AI Infrastructure Project

This is a learning project for AI infrastructure development, focusing on building practical skills in:
- C++ development with modern tooling
- Build systems (Bazel)
- Third-party library integration
- Testing and CI/CD practices

## Project Structure

```
ai_infra/
├── README.md
├── WORKSPACE
├── BUILD
├── src/
│   └── main.cpp
└── third_party/
    └── (external dependencies)
```

## Getting Started

```bash
# Clone the repository
git clone git@github.com:james20260418/ai_infra.git

# Build and run
bazel run //:main
```

## Goals
1. Learn modern C++ development practices
2. Master Bazel build system
3. Integrate common AI/ML libraries
4. Build reusable infrastructure components
